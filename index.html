About Start.ng

Start.ng is HNG pre-internship program tailored towards beginners, who wants to learn the basics of programming remotely. This is a great way to start honing your coding skills. Also, you can learn at your own pace, it doesn't matter whether you're working or a student, you can learn in the evening and it's totally free.

Read more

We help grow young talents

This platform will create a foundation to help you launch your career in coding. With a propelling belief that everyone should begin thier software journey from the easiest starting point and then work thier way up to the basic web language before moving on to complex aspects of coding.
